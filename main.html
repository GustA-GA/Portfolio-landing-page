Sobre mim

Sou um desenvolvedor apaixonado por computadores e tecnologias como um todo. Com uma grande base em HTML e CSS, meu foco é a criação de soluções e projetos que possuam tanto uma ótima funcionalidade quanto uma experiência de usuário de qualidade inolvidável.

No passado, durante meu estágio e minha graduação, tive a oportunidade de desenvolver uma variedade de projetos com diferentes requisitos que foram essenciais para incrementar meu repertório técnico, como dispositivos de IoT (do inglês Internet of Things) comandados por microcontroladores, aplicativos Android para coletas de dados em um jogo educacional infantil, além de um robô de controle remoto equipado com sensores para coletas de parâmetros como temperatura, umidade e índice de presença de gases no ambiente em que está operando.

Buscando novos desafios e fazer parte de um meio que o trabalho em equipe seja valorizado.

Projetos

  • RPG Character Creator

    O RPG Character Creator é uma aplicação web-based desenvolvida para auxiliar jogadores a criarem e manterem seus personagens de RPG. O usuário pode inserir informações sobre seu personagem, uma foto para o mesmo, distribuir pontos de atributos, equipar itens do inventário com a funcionalidade "drag n' drop" (arrastar e soltar), além de poder criar e adicionar novos itens.

    Utilizado nesse projeto: HTML CSS JavaScript

  • Ambient Monitoring Robot

    This project, called RMDA in portuguese, was designed to aid in monitoring and inspection routines in industrial environments, especially when there is a possibility of harmful gas leaks. While being controlled, the robot can also collect and display data such as gas concentration at its location, temperature, and humidity.

    HTML CSS JavaScript
